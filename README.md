# Sample

## Recommendations
- For the name of the solution, you need to use only Latin characters, numbers and dots.

## Usage
1. Clone this repo
2. Install template
```
cd src
dotnet new install .
```
3. Create new solution
- From cmd:
```
dotnet new stio_layered -n LayeredSolution
```
- From IDE  
Template name: **Stio Layered Template**
